# Jarvis AI Assistant

A simple voice assistant built with Python.

## Features
- Greets the user based on time
- Listens to voice commands
- Tells the current time

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3

## How to Run

1. Clone the repository
2. Install dependencies

pip install pyttsx3  
pip install SpeechRecognition  
pip install pyaudio  

3. Run the program

python jarvis.py
